# 🎓 Attendance Management System 

A modern **Attendance Management System** built with **Python Tkinter**, featuring a professional dark-themed GUI, SQLite database integration, CSV export, analytics dashboards, email reporting, and automated backups.

This project is designed for managing attendance records of students or employees with an intuitive desktop interface and real-time reporting features.

---

## ✨ Features

-  Modern Tkinter GUI with dark professional theme
-  Attendance marking system (Present / Absent / Late)
-  SQLite database integration
-  CSV export and synchronization
-  Dashboard with statistics and analytics
-  Attendance reports with charts (Matplotlib)
-  Member management system
-  Search and filter attendance records
-  Update and delete attendance entries
-  Email attendance reports with attachments
-  Automatic backup system
-  Responsive sidebar navigation
- ✅ Real-time clock and toast notifications

---

## 🛠 Technologies Used

- Python 3
- Tkinter
- SQLite3
- CSV
- Matplotlib
- SMTP Email System
- Threading

## 📧 Gmail App Password Setup

To use the email feature:

1. Enable **2-Step Verification** in your Google Account
2. Go to:
   - Google Account → Security → App Passwords
3. Generate an App Password
4. Use that password inside the application
